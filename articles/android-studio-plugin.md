# 環境

Mac OS 13.1（22C65）
Android Studio Dolphin | 2021.3.1

# 手順

1. Preferences > Plugin と選んで、左上の検索窓から検索する。
1. installをクリックして、installedになるまでしばらく待つ
1. Applyしてウィンドウを閉じる
1. Android Studioを再起動するか聞かれたら、再起動する
